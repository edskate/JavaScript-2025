Funções do Javascript: prompt

O prompt é uma função que exibe uma caixa de diálogo para o usuário inserir um valor.

Ele pode ser usado para coletar informações do usuário, como nome, idade, etc.

O valor inserido pelo usuário é retornado como uma string.

Exemplo:

  • A função promptrecebe um dado do usuário;
  • Podemos salvar este valor em uma variável;
    • Exemplo:
      const x = prompt("Digite um número:");
    1. uma função pouco utilizada, mas nos permite fazer ações interessantes;
